# Void-Start-VSOAPI

Void-Start-VSOAPI は、Void-Start-OS 上で動作するアプリケーション・コマンド向けの  
**公式 OS API セット** です。

Win32API のように、OS の内部機能を統一されたインターフェースとして公開し、  
ユーザーランド構築の基盤となることを目的としています。

---

# ✨ Features

- OS の機能を C++ API として提供  
- Console / Memory / Time / System の基本 API を実装  
- `#include <vsoapi.hpp>` だけで全 API にアクセス可能  
- 将来のユーザーランド開発を想定した拡張性の高い設計  
- Win32API を参考にした Result 型・ハンドル型の統一インターフェース
